# RAZ

RAZ is a modular pipeline toolkit designed for VFX, animation, and digital production workflows.

The goal of RAZ is to provide a flexible pipeline framework that can work in three modes:

- **Local** – for solo artists without a server
- **LAN** – for small studios
- **Server** – for distributed production environments

RAZ aims to combine ideas from ShotGrid Toolkit, AYON, and Prism while remaining lightweight, extensible, and open.

## Features (planned)

- Workfiles management
- Publish system
- DCC integrations (Houdini, Maya, Blender)
- SQLite registry for local projects
- Optional server backend
- Web UI for production tracking

## Status

Early development – architecture and core systems are being implemented.
